HTML Tags Ordered List

    List of HTML Tags

    1. <html>: The root element
    2. <head>: Contains document metadata
    3. <title>: Sets the page title
    4. <meta>: Metadata about the document
    5. <link>: External resource links
    6. <style>: Embedded CSS
    7. <script>: JavaScript code
    8. <body>: Page content
    9. <h1>: Heading 1
    10. <p>: Paragraph
    11. <a>: Hyperlink
    12. <img>: Image
    13. <ul>: Unordered List
    14. <ol>: Ordered List
    15. <li>: List Item
    16. <div>: Generic container
    17. <span>: Inline container
